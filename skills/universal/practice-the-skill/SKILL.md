# Practice The Skill

Train the target ability, not just knowledge about it. Start at the learner's current level, shorten feedback latency, and track proof of improvement.

## Process

1. Establish target ability, current evidence, motivation, time/energy, materials, friction, feedback source, and competence test.
2. Design one repeatable loop: focused input → attempt → feedback → corrected repetition → spaced review → progression gate.
3. Make the first session small enough to complete and repeat.
4. Save useful attempts, corrections, examples, recordings, projects, or reflections in the least-friction durable record.
5. Increase difficulty only when the evidence meets the progression gate.
